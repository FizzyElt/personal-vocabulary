# 個人詞彙紀錄

自己建立自己的英文詞庫，不要刻意去背你其實平常用不到的單字

目前單純以 JSON 檔案操作為主，未來有機會用 database。

## 功能
- server
  - [X] 獲取單字
  - [X] 新增單字
  - [X] 更新單字
  - [X] 刪除單字
  - [X] 單字觀看次數增加
- client
  - [X] 顯示單字
  - [X] 觀看單字詳細資訊
  - [X] 刪除單字
  - [X] 更新單字
